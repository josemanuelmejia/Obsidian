# Multiple linear regression models

Main assumptions:

1) Linearity: continuous variable
2) The conditional meaning dependence assumption states that the explanatory variables are independent of the error. Including theoretically relevant variables, and making sure that those independent variables aren't correlated with the error.
3) Observations in the sample are independent, time series aren't those observations are correlated. You also can correct the standard errors... there are ways to allow its correlation. The sample is normally distribution, if not you don't have the conditions for the regression.

Always run a Wald test.
